GLossário de Computadores de Redes

Glossário de computadopres de redes com 3 asssuntos importantes e seus termos.

IP
Um endereço IP (Endereço de Protocolo da Internet) é um identificador numérico único atribuído a cada dispositivo conectado a uma rede, como a internet ou uma rede local, para que possam se comunicar entre si de forma eficiente.

DNS
Em redes, o DNS (Domain Name System, ou Sistema de Nomes de Domínio) é um sistema essencial que traduz nomes de domínio, como www.exemplo.com, em endereços IP numéricos.

Roteador
Um roteador de internet é um aparelho que permite conectar vários dispositivos a uma rede e à Internet, atuando como um "direcionador" de dados.
