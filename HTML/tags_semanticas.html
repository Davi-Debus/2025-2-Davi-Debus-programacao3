O que é o HTML?

Pesquisa efetuada em 15/08/2025.

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de páginas web. Não é uma linguagem de programação, mas sim uma forma de organizar e apresentar informações (texto, imagens, etc.) para que os navegadores possam exibir corretamente. O HTML define a estrutura, o layout e o conteúdo de uma página web, enquanto outras tecnologias, como CSS e JavaScript, são usadas para estilização e interatividade.
